Directions: In Question No. 36-45, some parts of the given sentences have errors and some are correct. Find out which part of the sentence has an error and if a sentence is free from errors, your answer is (d) No error.

36. Josh and Jane (A) / are my (B) / family members (C) / No error (D)
a) A
b) B
c) C
d) D
Correct Answer: c) C (The correct phrasing is usually 'members of my family' or 'are my family')

37. Her friend did not see her (A) / for many years (B) / when she met her last night (C) / No error (D)
a) A
b) B
c) C
d) D
Correct Answer: b) B (Should be 'had not seen her' or 'had not seen each other')

38. Harry could not (A) / go to the party (B) / for his father was not well (C) / No error (D)
a) A
b) B
c) C
d) D
Correct Answer: d) D

39. There was a (A) / number of meetings (B) / which produced little of practical value (C) / No error (D)
a) A
b) B
c) C
d) D
Correct Answer: a) A (Should be 'There were a number of meetings' because 'a number of' takes a plural verb)

40. Neither Mr. Sharma nor Mr. Gupta (A) / were able to give (B) / Suresh any advice (C) / No error (D)
a) A
b) B
c) C
d) D
Correct Answer: b) B (Verb should agree with the nearer subject, 'Mr. Gupta', so it should be 'was able to give')

41. Nina is (A) / our cousin (B) / sister (C) / No error (D)
a) A
b) B
c) C
d) D
Correct Answer: c) C (The word 'sister' is redundant. 'Nina is our cousin' is correct.)

42. The children sat (A) / at the shade of (B) / the mango tree (C) / No error (D)
a) A
b) B
c) C
d) D
Correct Answer: b) B (Should be 'in the shade of')

43. If it will rain (A) / we shall (B) / not attend the football match (C) / No error (D)
a) A
b) B
c) C
d) D
Correct Answer: a) A (In a first conditional sentence, the 'if' clause must use simple present: 'If it rains')

44. We respect her (A) / as she is (B) / our Principal (C)/No error (D)
a) A
b) B
c) C
d) D
Correct Answer: d) D

45. None of the rooms (A) / are available for occupation (B) / we were told by the manager (C)/No error (D)
a) A
b) B
c) C
d) D
Correct Answer: b) B (The pronoun 'None' generally takes a singular verb, so 'is available' is preferred over 'are available')
