Directions: In Q. Nos. 1-5, some of the sentences have errors and some have none. Find out which part of a sentence has an error, i.e., (A), (B) or (C). Mark (D) if the sentence has no error.

1. The pupil respectfully asked / the teacher if he / could went in. / No error
   (A) / (B) / (C) / (D)
   Answer: (C) (Error: should be 'could go in'.)

2. The beggar thanks / him a lot / for the help. / No error
   (A) / (B) / (C) / (D)
   Answer: (A) (Error: should be 'The beggar thanked' or 'The beggar thanks' if a generic statement, but 'thanked' is contextually better.)

3. The War of Kalinga proved / to be a turning point / in the live of Ashoka, the Great. / No error
   (A) / (B) / (C) / (D)
   Answer: (C) (Error: 'live' should be 'life'.)

4. Democracy is a Government by the people / off the people / and for the people. / No error
   (A) / (B) / (C) / (D)
   Answer: (B) (Error: 'off' should be 'of'.)

5. Johny ran fast / lest he should be / late for the class. / No error
   (A) / (B) / (C) / (D)
   Answer: (D) (No error.)
